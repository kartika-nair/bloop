# Bloop!

Bloop is an application that helps you go offline when you've been doomscrolling for too long (even though you said it would only be a couple of minutes). Because we believe that touching grass shouldn't feel like a chore, and should instead feel like levelling up :)

Created for the Women in ECS Hackathon 2026 by Kartika Nair, Shree Kriti Ramesh, Deyasini Goswami, and Isha Kulkarni.
